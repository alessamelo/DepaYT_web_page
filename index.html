Kitchen Cleanliness Prediction using CNN

Deep Learning Project using NASNet + MLP Classifier

Project Overview

This project uses Convolutional Neural Networks (CNN) and Transfer Learning with NASNetLarge to classify kitchen images as clean or dirty. Images are collected from Bing and manually captured from Yachay Tech residences.

The extracted CNN features are classified using an MLP neural network to improve prediction performance.
